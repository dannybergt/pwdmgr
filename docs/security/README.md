# Security Documentation

Security is a design constraint, not a late-stage audit activity.

Baseline references:

- OWASP ASVS 5.0.0 target level 2 for MVP, level 3 for critical modules.
- OWASP Password Storage Cheat Sheet.
- OWASP Secrets Management Cheat Sheet.
- OWASP Docker Security Cheat Sheet.
- NIST SP 800-63-4.
- NIST Cybersecurity Framework 2.0.
- OWASP GenAI/LLM Top 10 for prompt-injection and agentic AI risks.

Required security artifacts:

- Threat model.
- Crypto design.
- Tenant isolation tests.
- Secure coding guidelines.
- Dependency and image scanning.
- Agent signing and update design.
- Incident response procedures.
